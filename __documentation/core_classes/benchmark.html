Becnhmark Class

The Benhmark class is used to see how fast certain components are taking to load, as well as how much memory your script is using at any given point.

Loading the Config class

If you need to access the function manually, you will do so like below. Please note that load_class() is the default way of loading Core Classes... but you can use the loader.

$this->Benchmark = load_class('Benchmark');

Starting a new Benchmark

To initiate a new start time:

$this->Benchmark->start('$name');
Arguments:

$name [Type: String]: the name of your benchamrk, but remember it, you will need this later ;)

Stoping a Benchmark

To initiate a a stop time to your benchmark:

$this->Benchmark->stop('$name');
Arguments:

$name [Type: String]: the name of your benchamrk

Getting a Benchmark time

This method will fetch the elapsed the time in seconds it took from start to stop.

$this->Benchmark->elapsed_time('$name', '$round', '$stop');
Arguments:

  1. $name [Type: String]: the name of your benchamrk
  2. $round [Type: Int, Default: 3]: how many numbers after the decimal point? (Ex: 3 would look like this - 0.123)
  3. $stop [Type: Bool, Default: FALSE]: Do we stop the benchmark upon showing it?

Displaying Memory Usage

This method shows how much memory usage the script is taking up at that current moment

$this->Benchmark->memory_usage();
