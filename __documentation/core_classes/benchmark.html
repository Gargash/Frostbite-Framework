Becnhmark Class

The Benhmark class is used to see how fast certain components are taking to load, as well as how much memory your script is using at any given point.

Starting new Benchmark

To initiate a new start time:

Benchmark::startTimer('$name');

$name: the name of your benchamrk, but remember it, you will need this later ;)

Stoping a Benchmark

To initiate a a stop time to your benchmark:

Benchmark::stopTimer('$name');

$name: the name of your benchamrk

displaying a Benchmark time

Please take note, if you do not stop your timer first, It Will Be Stopped after using this function.

Benchmark::showTimer('$name');

$name: the name of your benchamrk
