Session Class

The session class makes it easier to manage users login session information, and is more secure as it does not use the $_SESSION global. All session information is stored in the database for easy and secure reach. If you have the session Database turned off, then custom information can be stored in the cookie instead, which is WAY less secure then storing info in the database.
